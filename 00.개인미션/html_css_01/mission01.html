CSS 배치 이해하기

CSS의 display 속성은 HTML 요소가 페이지에 어떻게 나타날지를 결정하는 데 사용됨.

✅ 개념 설명

display: none

  • 요소를 보이지 않게 숨기며 공간도 차지하지 않음.
  • DOM에는 남아 있지만 사용자에게는 보이지 않음.

display: block

  • 요소를 블록 요소처럼 배치.
  • 전체 가로 폭을 차지하며, 줄 바꿈이 자동으로 일어남.

display: inline

  • 요소를 인라인 요소처럼 배치.
  • 컨텐츠만큼의 너비를 차지하며 줄 바꿈이 일어나지 않음.
