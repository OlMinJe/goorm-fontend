시맨틱 태그 이해

✅ 1. 시맨틱 태그란?

HTML5에서 도입된 시맨틱(semantic, 의미 기반) 태그는 웹 페이지의 구조를 명확히 설명하는 태그로, 개발자/검색엔진/보조 기술(스크린리더 등) 모두에게 페이지 구조를 쉽게 이해시키는 데 도움됨.

✅ 2. 주요 시맨틱 태그 소개

태그 설명
<header> 사이트나 섹션의 헤더 영역 (로고, 내비게이션 포함)
<nav> 내비게이션 링크를 담는 영역
<main> 문서의 핵심 콘텐츠 영역 (하나의 문서에 하나만 사용)
<section> 관련 있는 콘텐츠를 묶는 구획 영역
<article> 독립적으로 의미를 가지는 콘텐츠 (예: 뉴스, 상품 정보 등)
<aside> 보조 콘텐츠 (광고, 관련 링크 등)
<footer> 하단 영역, 저작권, 연락처, 이용약관 등
