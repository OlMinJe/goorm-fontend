# 📘 배열

## 📌 배열이란?

- 번호(인덱스)와 데이터로 이루어진 자료 구조

인덱스는 항상 0부터 시작

자바스크립트의 배열은 Array 생성자 함수로 만든 객체(Object)

### 배열 생성 방법

1. 리터럴 표기법 → 가장 일반적, 간결, 직관적

```javscript
const nums = [1, 2, 3];
const mixed = ["이름", 123, true];
```

2. `new Array()` 생성자

- `new Array(1, 2, 3)` → `[1, 2, 3]`
- `new Array(3) → 길이가 3인 빈 슬롯 배열

## 📌 배열의 주요 메서드

### 추가/제거

- `push()` / `pop()` → 끝에 추가/제거
- `unshift()` / `shift()` → 앞에 추가/제거

### 순회/변환

- `forEach()` → 반복 실행
- `map()` → 변환 후 새 배열 반환
- `filter()` → 조건에 맞는 요소만
- `find()` → 조건 만족 첫 요소 반환
- `some()` / `every()` → 일부/전부 조건 검사

### 정렬

- `sort()` → 기본은 문자열 기준, 숫자는 비교 함수 필요
  예: `(a, b) => a - b`
- reverse() → 역순

### 추출/변형

- `slice()` → 잘라서 새 배열
- `splice()` → 중간 삽입/삭제
- `join()` → 문자열로 합침
- `flat()` → 중첩 배열 펼침
- `at()` → 음수 인덱스 가능
- `reduce()` → 누적 계산

## 정리

- 자바스크립트 배열은 객체이므로 다양한 메서드와 속성을 가짐
- 숫자 정렬 시 `sort()`에 반드시 비교 함수를 넣어야 함
- `new Array(3)`과 `[undefined, undefined, undefined]`의 차이 이해
- `forEach`, `map`, `filter` 등 고차 함수의 활용법 익힘
