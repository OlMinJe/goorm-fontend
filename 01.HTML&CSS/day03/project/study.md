# CSS 우선 순위

| 순위 | 우선순위 기준                    | 예시                             |
| ---- | -------------------------------- | -------------------------------- |
| 1    | `important` 선언                 | `color: red !important;`         |
| 2    | 인라인 스타일                    | 요소 내 스타일 속성              |
| 3    | ID 선택자                        | `#header` 등                     |
| 4    | 클래스(class), 속성, 가상 클래스 | `.menu`, `[type=text]`, `:hover` |
| 5    | 요소(tag) 선택자                 | `div`, `p` 등                    |
| 6    | 브라우저 기본 스타일             | `<h1>` 태그 기본 스타일 등       |
